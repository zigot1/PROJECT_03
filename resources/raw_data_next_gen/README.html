BIM 360 Data Connector

Overview

A common feature request for Autodesk's BIM 360 solution has been the ability to connect Business Intelligence (BI) tools to analyze and process the data created in the various BIM 360 offerings. The BIM 360 Data Connector is the answer for this requirement. The Data Connector allows users to request and download data housed in the BIM 360 Data Platform in a well defined schema/format. This data can then be used for further analysis and to drive customer specific workflows.

Data Schema & Design

A central aspect to the Data Connector is the data schema for BIM 360 data extracts. The data is extracted into a well defined schema much in the way a traditional relational database schema is designed. Each service group is a separate "database" schema design and each individual table in this design will be delivered via separate CSV file extracts. When a data request is completed, the Data Connector will generate a ZIP archive file that will be made available for download by the user. The naming conventions and directory structure for ZIP file extracts are the following:
  • autodesk_data_extract.zip
    • <schema1>_<table_name1>.csv
    • <schema1>_<table_name2>.csv
    • <schema1>_<table_name3>.csv
    • <schema2>_<table_name1>.csv
    • <schema2>_<table_name2>.csv
    • <schema2>_<table_name3>.csv
    • etc...
    • docs <directory>
      • README.html - this document
      • schemas <directory>
        • <schema>.html - detailed schema documentation
